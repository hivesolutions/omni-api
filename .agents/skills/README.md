# Agent Skills

These provide instructions for common tasks:

| Skill           | Description                                                                                                                                              |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `add-api-types` | Add or complete `.pyi` type stubs for API operations: read entity types, `Delta`/`Payload` write types, hierarchy alignment and live wire-format checks |

Each skill has a `SKILL.md` with instructions and supporting files like `examples.md` and `template.md`.
